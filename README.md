#Layoffs Data Analysis Project :
# Overview:
This project analyzes global layoffs data using SQL to uncover trends, patterns, and insights. It focuses on data cleaning, transformation, and exploratory data analysis (EDA) to answer key business questions.

# Objectives:
Clean and standardize raw layoffs data
Handle missing and inconsistent values
Perform exploratory data analysis
Identify trends by date, company, industry, and country

# Tools & Technologies:
SQL (MySQL)
Excel (for initial data review)
Database concepts (CTE, Window Functions, Aggregations)

# Data Cleaning Steps:
Removed duplicates
Standardized date formats
Handled NULL values
Fixed inconsistent text values (company names, industries)

# Key Analysis Performed:
Total layoffs over time
Monthly and yearly trends
Top companies with highest layoffs
Layoffs by industry and country
Rolling totals using window functions

# Key Insights:
Layoffs peaked during specific economic periods
Certain industries were more affected than others
A few companies contributed heavily to total layoffs

# How to Use:
Import the dataset into MySQL
Run the SQL script (project_layoff.sql)
Analyze outputs and modify queries as needed

# Future Improvements:
Create dashboards using Power BI / Tableau
Automate data pipeline
Add predictive analysis
